# Information Extraction from Turkish Radiology reports based on DyGIE++

## Info
Data is in `data/radtr` directory. The fine-tuned model file is in google drive because it is large [radtr-BioBERTurk-801](https://drive.google.com/file/d/1DiOaFZsIxqXHfHzFxJcKjpvPve2lq8bf).

The actual code execution is in `main.ipynb`

## Setup
- Use python version 3.7
- Use latest version of pip 24.0
- Create virtual environment: `python -m venv .venv`
- Activate virtual environment: `source ./venv/bin/activate`
- Install dependencies: `pip install -r requirements.txt`

## Execution
- Run the code in main.ipynb
